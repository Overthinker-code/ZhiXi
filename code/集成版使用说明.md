# 课堂行为检测 - 完整集成版使用说明

## 🎯 功能概述

这是一个**两阶段**的课堂行为检测方案：

```
教室图片 
    ↓
[阶段1] YOLO检测器 → 框出所有学生位置
    ↓
[阶段2] YOLO-Pose → 对每个框提取姿态关键点
    ↓
[阶段3] 行为识别 → 判断每个学生行为（专注/手机/睡觉等）
    ↓
返回：每个人位置 + 行为 + 整体统计
```

---

## 🚀 快速启动

### 1. 基础启动（快速模式 - 推荐）

```bash
# 无需队友的检测模型，直接使用YOLOv8-Pose
python yolo_integrated.py

# 或
uvicorn yolo_integrated:app --port 8000
```

### 2. 双模型模式（如果队友有训练好的检测器）

```bash
# 假设队友的检测模型在 yolo_test/best.pt
export DETECTOR_MODEL_PATH="yolo_test/best.pt"
export DETECTION_MODE="dual"

python yolo_integrated.py
```

## 🔧 与队友的 yolo_test 配合

### 场景1：队友只提供了测试数据（yolo_test）

**现状：**
- `yolo_test/` 有8张测试图
- 没有训练好的检测模型（.pt文件）

**解决方案：**
直接使用**快速模式**（无需额外模型）：
```bash
# YOLOv8-Pose 自带检测能力，可以直接框出人并估计姿态
uvicorn yolo_integrated:app --port 8000
```

### 场景2：队友提供了训练好的检测模型

**假设队友有：**
- `yolo_test/best.pt` 或 `yolo_test/yolov8n.pt`

**启动方式：**
```bash
# Linux/Mac
export DETECTION_MODE="dual"
export DETECTOR_MODEL_PATH="yolo_test/best.pt"
python yolo_integrated.py

# Windows PowerShell
$env:DETECTION_MODE="dual"
$env:DETECTOR_MODEL_PATH="yolo_test/best.pt"
python yolo_integrated.py

# Windows CMD
set DETECTION_MODE=dual
set DETECTOR_MODEL_PATH=yolo_test/best.pt
python yolo_integrated.py
```

### 场景3：需要用 yolo_test 的数据训练检测器

如果队友的8张图是用来**训练**的，需要更多数据：

```bash
# 1. 收集更多教室场景图片（至少100张）
# 2. 用 yolo_test 的格式标注（YOLO格式）
# 3. 训练检测模型

yolo detect train \
  data=yolo_test/obj.data \
  model=yolov8n.pt \
  epochs=100 \
  imgsz=640 \
  project=yolo_test \
  name=trained

# 4. 训练好的模型会在 yolo_test/trained/weights/best.pt
# 5. 用这个模型启动集成服务
export DETECTOR_MODEL_PATH="yolo_test/trained/weights/best.pt"
python yolo_integrated.py
```

## 🎨 前端展示建议

后端返回的数据包含每个人位置和行为，前端可以：

```javascript
// 示例：在画面上绘制结果
const result = await fetch('/analyze/classroom', {method: 'POST', body: formData});
const data = await result.json();

// 绘制每个人
 data.persons.forEach(person => {
   // 画框
   drawRect(person.bbox.x1, person.bbox.y1, person.bbox.x2, person.bbox.y2, 
            person.behavior_result.color);
   
   // 画标签
   drawText(person.bbox.x1, person.bbox.y1 - 10, 
            `${person.behavior_result.behavior} (${person.behavior_result.confidence})`);
 });
 
 // 显示统计
 showStats(data.classroom_stats);
```

## 📈 准确率优化建议

### 当前方案准确率

| 方案 | 准确率 | 适用场景 |
|------|--------|----------|
| 快速模式 + 规则 | 60-70% | 紧急使用，无需准备 |
| 快速模式 + LSTM | 75-85% | 有训练数据时 |
| 双模型 + LSTM | 80-90% | 队友有检测模型时 |

### 提升准确率的方法

1. **收集行为识别训练数据**
   ```bash
   # 录制单人行为视频，放入 data/ 目录
   python quick_collect_and_train.py
   ```

2. **调整检测阈值**
   ```bash
   export CONF_THRESHOLD=0.7  # 提高置信度阈值，减少误检
   ```

3. **使用更大的姿态模型**
   ```bash
   # 下载 yolov8s-pose 或 yolov8m-pose
   yolo download yolov8s-pose
   export YOLO_MODEL_PATH="yolov8s-pose.pt"
   ```

## ❓ 常见问题

### Q1: 检测不到人？
**A:** 检查图片质量，调整置信度阈值：
```bash
export CONF_THRESHOLD=0.3  # 降低阈值
```

### Q2: 检测到人但行为识别失败？
**A:** 可能是姿态被遮挡，检查 `successfully_analyzed` 字段

### Q3: 如何配合队友的 yolo_test？
**A:** 先问队友：
- 有没有训练好的 `.pt` 模型文件？
- 还是只有8张测试图？
- 需不需要用这8张图训练？

### Q4: 只能检测不能识别行为？
**A:** 检查日志，看行为识别模块是否加载成功（LSTM或规则）
